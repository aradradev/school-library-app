# 📖 [School Library App] <a name="about-project"></a>

**[School Library App]** is an application that holds school library information. The application enables the one to add a new student, a teacher or a book and also store information of whoever borrows a book and when.

## 💻 Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps.

### Prerequisites
In order to run this project you need:

- Installation of Ruby
- Terminal
- VS code editor 

### Setup

Clone this repository to your desired folder:

```sh
  cd school-library-app
  git clone https://github.com/aradradev/school-library-app.git
```

Run the main file to test the different classes:

```sh
ruby ./main.rb
```
